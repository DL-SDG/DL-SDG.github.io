******************
DL_FIELD
******************

This is the main portal to carry out DL_FIELD exercises. If you want to get familiarise certain program features, you can refer to :doc:`../TUTORIALS/dlf_0` reference guide.

Note that by carrying out DL_FIELD exercises does not make you an expert user overnight. They only provide a flavour how DL_FIELD functions and operates. This is because DL_FIELD is rich in features that can be combined in variety of ways depending on your needs and models.
