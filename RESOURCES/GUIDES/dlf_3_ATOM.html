An ATOM is a DL_FIELD-specific jargon. It refers to an entity, or member of a MOLECULE. An ATOM can be a normal atom, as in a normal molecule, an ion, a pseudo point, core or shell part of an atom, united atom or coarse-grain particle.

For a given FF scheme, all the available ATOMs are listed within the ATOM_TYPE and END ATOM_TYPE directives in the sf files. These ATOMs will then be used to construct a MOLECULE template.
